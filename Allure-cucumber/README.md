# Allure Cucumber Example
Using Allure with cucumber JVM

# Target 
a simple calculator with allure report

# Testing commands  

### Running Tests 
(use maven 3.3.9)

- Run test ```mvn clean test``` 

### Running Report
 - Build Site ```mvn site```
 - Run Site ```mvn jetty:run```

See in your http://localhost:9100 

# Requirements 
- Updated to JDK 21
